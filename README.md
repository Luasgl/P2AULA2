# Projeto - Fundamentos Python 2 Aula 2
Repositório com projeto desenvolvido em aula pelo curso do Python Essentials 2 oferecido pela Cisco Networking Academy através do Programa Women Rock IT Brasil (WRIT) 2025.
Programa de celebração e empoderamento de mulheres na insdústria de tecnologia (STEM). Uma jornada inspiradora de networking e aprendizado com nossos incríveis professores e parceiros, com o objetivo de incentivar e apoiar as mulheres na busca por uma carreira em tecnologia.
